# Typography Pairings Reference

15 validated font pairings from the bundled collection. Each pairing includes weight guidance, letter-spacing, and a modular type scale.

---

## Type Scale System

All pairings use one of two modular scale ratios:

| Ratio | Name | Progression (base 16) |
|---|---|---|
| 1.250 | Major Third | 16 / 20 / 25 / 31 / 39 / 49 / 61 / 76 |
| 1.333 | Perfect Fourth | 16 / 21 / 28 / 38 / 50 / 67 / 89 / 119 |

Scale steps: caption / body / subheading / heading-3 / heading-2 / heading-1 / display-2 / display-1

## Pairing 1: Editorial Luxe

> **Display:** InstrumentSerif (Regular, Italic)
> **Body:** Outfit (Light 300, Regular 400)

**Mood:** High-end editorial, fashion, gallery. Sophisticated restraint.
**Scale:** Perfect Fourth (1.333)
**Letter-spacing:** InstrumentSerif headings: -0.02em. Outfit body: 0.01em.
**Weight guidance:** InstrumentSerif has no bold -- use italic for emphasis. Outfit Light for body, Regular for UI elements.
**Best for:** Portfolio covers, art exhibition posters, luxury brand content.

## General Typography Rules

1. **Maximum 2 font families per design.** A display + body pairing is sufficient. Adding a third font requires strong justification.
2. **Create hierarchy through weight and size, not additional fonts.** Use Bold, Regular, Light within one family before reaching for another.
3. **Minimum body text size:** 16px on screen, 10pt in print. Never go smaller for readable content.
4. **Line height for body text:** 1.4 to 1.6 relative to font size. Tighter for headings (1.0 to 1.2).
5. **Maximum line length:** 60-75 characters for comfortable reading. Roughly 600px at 16px body text.
6. **Caps lock text** requires increased letter-spacing (0.08em to 0.15em minimum). Never set long text in all caps.
7. **Italic** is for emphasis, titles, or foreign words. Not for decoration. Script fonts replace italic -- never combine them.
