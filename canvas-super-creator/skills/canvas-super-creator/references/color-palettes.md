# Color Palettes Reference

20 curated palettes with defined roles and contrast guidance. Every palette is tested for WCAG AA compliance on at least one text/background combination.

---

## How to Read Each Palette

| Role | Purpose |
|---|---|
| **background** | Primary canvas fill |
| **foreground** | Primary text and line art |
| **accent-primary** | Key interactive or focal element |
| **accent-secondary** | Supporting accent, complementary to primary |
| **muted** | Subdued backgrounds, dividers, secondary text |
| **highlight** | Emphasis, callouts, hover states |

## FORBIDDEN Color Patterns

Do not use these in any design. They instantly signal generic AI output or amateur work.

| Pattern | Why It Fails |
|---|---|
| Purple-to-blue linear gradient as primary background | Overused by AI tools. Ubiquitous in generic SaaS. Signals "default". |
| Rainbow gradient or spectrum stripes | No design intent. Visual noise. Reads as clip art. |
| Neon green (#00FF00) on black | Matrix cliche. Unreadable at small sizes. Retinal burn. |
| Default Midjourney/DALL-E palettes (oversaturated teal + magenta) | Immediately recognizable as AI-generated. |
| Uniform pastel rainbow (equal-weight pastel colors in sequence) | No hierarchy. No focal point. Reads as a children's toy. |
| Pure white (#FFFFFF) text on pure black (#000000) background without adjustments | Halation effect causes eye strain. Use off-white on near-black instead. |
| Any palette with more than 6 colors used equally | No hierarchy. Visual chaos. Stick to defined roles. |
