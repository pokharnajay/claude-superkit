## Tips for Mesh Gradients

### Controlling Density
- More layers = smoother blends but heavier rendering
- 4-6 layers is a good balance; 8 for hero backgrounds
- Use smaller `transparent` stop percentages (30-40%) for tighter color spots
- Use larger (50-60%) for diffuse washes

### Avoiding Color Banding
```css
/* Add a subtle noise overlay to break up banding */
.mesh-bg::after {
  content: '';
  position: absolute;
  inset: 0;
  background-image: url("data:image/svg+xml,%3Csvg viewBox='0 0 256 256' xmlns='http://www.w3.org/2000/svg'%3E%3Cfilter id='n'%3E%3CfeTurbulence type='fractalNoise' baseFrequency='0.7' numOctaves='4' stitchTiles='stitch'/%3E%3C/filter%3E%3Crect width='100%25' height='100%25' filter='url(%23n)' opacity='0.08'/%3E%3C/svg%3E");
  background-size: 256px 256px;
  opacity: 0.15;
  pointer-events: none;
  mix-blend-mode: overlay;
}
```

### Combining with Background Color
Always end your gradient stack with a `linear-gradient` or solid color as the final fallback layer. This ensures no white gaps appear.

### Animation-Ready Pattern
You can animate mesh gradients by changing the position of the radial centers:
```css
.animated-mesh {
  background-size: 200% 200%;
  animation: mesh-shift 10s ease-in-out infinite;
}
@keyframes mesh-shift {
  0%, 100% { background-position: 0% 0%; }
  50% { background-position: 100% 100%; }
}
```

### Generating Custom Meshes
1. Pick 4-8 colors from your palette
2. Place them at different x/y positions (avoid clustering)
3. Use `ellipse` shape for organic feel, `circle` for uniform
4. Set opacity between 0.4-0.9 (higher for key colors)
5. Set transparent stop between 35-55%
6. Add a linear-gradient base layer using your darkest color
