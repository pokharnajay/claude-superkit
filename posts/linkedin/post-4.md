# Post 4 - I Taught AI to Design
**Series:** Building with Claude · Part 4 of 5
**Graphic:** assets/linkedin-post-4.png (1280×720)

---

Data was covered. Automation was covered.

But there was one thing I still couldn't do - make Claude create something *beautiful*.

I tried every Python image library I could find. All of them. All bad.
Flat colours. Broken layout. Output that looked like clip art from 2008.

Then one day I had a stupid-simple idea.

What if instead of drawing pixels - I just wrote HTML and CSS?

CSS already knows how to do everything I wanted.
Glassmorphism. Gradient orbs. Blend modes. Gradient text. Clip-path shapes. Texture overlays.
And Playwright can screenshot any webpage at any resolution.

The quality difference was immediate and dramatic.

So I kept going. What started as a rendering trick became a full design system.

→ 8 skills covering every design category
→ 5 specialist agents - layout, typography, colour, rendering
→ 80+ bundled fonts, loaded locally - no internet needed
→ Output types: covers, posters, social posts, thumbnails, brand assets, abstract art

That's **Canvas Design**.
Born from frustration. Built from a single insight.
