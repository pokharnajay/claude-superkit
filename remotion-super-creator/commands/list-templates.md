Display the available video type templates:

| Template | Command | Description |
|----------|---------|-------------|
| Social Media | `/quick-social` | TikTok, Reels, Shorts, Stories (vertical 9:16) |
| Explainer | `/quick-explainer` | Product demos, tutorials (horizontal 16:9) |
| Kinetic Typography | `/quick-kinetic` | Text-driven motion graphics |
| Data Visualization | — | Animated charts, dashboards, data stories |
| Slideshow | — | Photo montages, portfolios, Ken Burns |
| Audiogram | — | Podcast waveforms, audio visualizations |
| Intro/Outro | — | Logo reveals, channel intros, end screens |
| News Highlight | — | Breaking news, headlines, tickers |

Reusable components in `templates/components/`: animated-text, progress-bar, particle-system, gradient-background, scene-wrapper, lower-third, countdown-timer, split-screen.
