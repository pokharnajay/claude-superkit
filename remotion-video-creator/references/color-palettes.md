# Color Palettes for Video Production

26 curated color palettes optimized for video. Each palette has 6 colors with defined roles: Background, Foreground/Text, Accent Primary, Accent Secondary, Muted/Subtle, and Highlight/Pop.

All hex values are chosen for high contrast ratios, good compression survival (avoiding near-threshold colors), and readability at 1080p and above.

## Quick Usage in Remotion

```tsx
// Define a palette as a constant
const palette = {
  bg: '#0B1120',
  fg: '#F0F4FC',
  accent1: '#3B82F6',
  accent2: '#60A5FA',
  muted: '#1E293B',
  pop: '#38BDF8',
};

// Use in components
<AbsoluteFill style={{ backgroundColor: palette.bg }}>
  <h1 style={{ color: palette.fg }}>Title</h1>
  <p style={{ color: palette.accent2 }}>Subtitle</p>
  <div style={{ backgroundColor: palette.accent1, color: palette.fg }}>
    CTA Button
  </div>
</AbsoluteFill>
```
